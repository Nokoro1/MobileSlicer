# Security Policy

## Supported Versions

Security reports are accepted for the current MobileSlicer beta and the current
`main` branch.

## Reporting A Vulnerability

Email security reports to <mobileslicerapp@gmail.com>.

Please include:

- MobileSlicer version or commit
- Android device model and Android version
- Printer model and firmware, if the issue involves printer communication
- Clear reproduction steps
- Relevant logs or screenshots, with private information removed

Do not post active security issues publicly until they have been reviewed.

## Credentials And Private Data

Do not include API tokens, printer passwords, Wi-Fi credentials, private model
files, or account credentials in public issues. If a secret is exposed, revoke or
rotate it before sending a report.

## Updates

Security fixes for beta builds are distributed through Google Play beta updates.
