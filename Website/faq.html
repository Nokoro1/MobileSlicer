FAQ

Before you install.

Short answers about MobileSlicer on Android.

Where do I download MobileSlicer?

MobileSlicer beta builds are distributed through GitHub Releases.

Is MobileSlicer available for iOS?

MobileSlicer is currently Android-only. iOS can be considered after the Android release is stable.

Is MobileSlicer free?

Yes. MobileSlicer is free, open source, and ad free.

Do I need to create an account?

No MobileSlicer account is required. GitHub may ask you to sign in only for GitHub-specific actions such as starring the repository or opening issues.

Why use GitHub Releases?

GitHub Releases provide the official beta APK, source snapshot, and release notes in one place.

Does slicing happen on-device?

Yes. Models are prepared, sliced, previewed, and exported locally on your Android device.

Does MobileSlicer upload my models?

No. Model loading, slicing, preview, and file export happen locally unless you choose to share or send a file.

What Android version is required?

The app currently targets Android 8.0 or newer, which is API 26 and up.

Which model formats work right now?

MobileSlicer supports STL import and 3MF mesh import for supported model/project files. Full desktop-Orca 3MF project metadata parity is still being expanded.

Which printers are supported?

MobileSlicer starts with curated printer profiles and supported network printer connections where available. More printer coverage will follow real user feedback.

Can I use it on a tablet?

Yes. The interface is designed for phones and tablets, with larger touch targets and mobile-first layouts.

Can MobileSlicer print directly to my printer?

Yes, where supported network printer connections are available. You can also export or share the generated print file.

How do I update to a newer version?

Download the newest APK from GitHub Releases and install it over the previous beta build.

What should I try first?

Import, workspace controls, profile selection, slicing, G-code preview, export or share, and saved plates are the most useful areas to try first.

Can I get MobileSlicer on Google Play?

Not right now. Current beta builds are distributed through GitHub Releases.
