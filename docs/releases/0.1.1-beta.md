# MobileSlicer 0.1.1 Beta

MobileSlicer 0.1.1 Beta is a release-hardening update for Android. It focuses
on Android 16 KB page-size compatibility, release APK verification, public beta
packaging, and tighter network-printer validation.

## APK

- Asset name: `MobileSlicer-0.1.1-beta-v4.apk`
- Application ID: `com.mobileslicer`
- Version name: `0.1.1-beta`
- Version code: `4`
- Minimum Android version: Android 8.0 / API 26
- Target SDK: 35
- APK SHA-256: `65616f78804155229fef808c05f1bf79281a0c27a667d9ba2ff602425a4472fc`

## Highlights

- Fixed Android 16 KB page-size compatibility for the release APK.
- Added a public release gate that installs and launches the signed release APK
  without relying on debug-only automation.
- Kept the experimental scanner entry out of release builds.
- Tightened Bambu LAN validation so connection tests and upload paths require
  the same host, 8-character access code, and device serial/device ID.
- Cleaned up Thingiverse naming and release-facing copy.
- Added the public website files to the repository while keeping local deploy
  state, signing files, SDKs, build output, APKs, and generated artifacts out of
  git.

## Verified

- Debug and release unit tests.
- Android release lint and release packaging.
- Python release/script tests.
- Website OAuth tests.
- Thingiverse release-readiness automated checks.
- Scanner synthetic regression and real-scan replay.
- Release worktree audit with no uncategorized paths.
- Signed release APK install and cold launch on a physical Android device.
- Release APK non-debuggable check.
- Release UI scanner-entry-hidden check.
- APK signature verification.
- 16 KB zip alignment verification.

## Known Release Notes

- Thingiverse OAuth backend code and tests are ready, but production deployment
  must still expose the `/v1/thingiverse/oauth/start` route before advertising
  Thingiverse import publicly.
- Bambu LAN validation is hardened in code, but public claims should wait for a
  real-printer smoke test covering upload-only, upload-and-start, bad access
  code, and offline-printer behavior.
